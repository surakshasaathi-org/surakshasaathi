# @suraksha/web-support

In-house CX console. Unified inbox (WhatsApp / email / web chat), customer 360, AI co-pilot.

## Layout

- **Inbox** — 3-column: conversation list | thread | co-pilot drafts
- **Customers** — search + customer 360 view
- **Macros** — reply snippets, glossary-aware
- **Co-pilot settings** — configure the `support-drafter` agent

## Ports

- Dev: `http://localhost:3002`
- Production: `support.surakshasaathi.com`
